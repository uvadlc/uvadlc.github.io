Deep Learning

Deep learning (or deep structured learning, or hierarchical learning or deep machine learning or simply neural networks) is a branch of machine learning based on a set of algorithms that attempt to model high-level abstractions in data by using multiple processing layers with complex structures, or otherwise composed of multiple non-linear transformations.
